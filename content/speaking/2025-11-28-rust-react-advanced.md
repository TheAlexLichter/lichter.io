A Vue/Nuxt person on stage at React Advanced - but the story is bigger than any single framework.

In this talk at React Advanced London 2025, I took React developers on a tour through the Rust-powered toolchain that's quietly becoming the foundation of modern frontend development: Rolldown as a faster bundler, Oxc as the underlying toolchain, Oxlint as an ESLint alternative that's 50-100× faster - all directly applicable to React projects today and setting the stage for the next generation of the React build pipeline.
